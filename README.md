# ComfyUI Kling Video-to-Audio (WaveSpeed API)

Кастомная нода для ComfyUI, которая генерирует звуковые эффекты и фоновую музыку для видео
с помощью модели Kling AI Video-to-Audio через WaveSpeed API.

## Возможности

- **Sound Effect Prompt** — текстовое описание звуковых эффектов (до 200 символов)
- **BGM Prompt** — текстовое описание фоновой музыки (до 200 символов)
- **ASMR Mode** — режим для усиления детальных звуковых эффектов
- Две ноды: для локального файла и для URL

## Требования

- Python 3.10+ (стандартная библиотека, доп. зависимостей нет)
- API-ключ WaveSpeed: https://wavespeed.ai/accesskey

## Установка

1. Скопируйте папку `ComfyUI_kling_video_2_audio` в `ComfyUI/custom_nodes/`
2. Перезапустите ComfyUI

## Ноды

### Kling Video to Audio (File)
- **video_path** — путь к локальному видео (mp4, до 20 сек)
- **wavespeed_api_key** — ваш API-ключ
- **sound_effect_prompt** — описание звуковых эффектов
- **bgm_prompt** — описание фоновой музыки  
- **asmr_mode** — вкл/выкл ASMR
- **Выход:** путь к видео с аудио (STRING)

### Kling Video to Audio (URL)
- **video_url** — публичный URL видео
- Остальные параметры аналогичны
- Пропускает шаг загрузки — быстрее, если видео уже в сети

## Использование

1. Найдите ноду: правый клик → `video/audio` → `Kling Video to Audio`
2. Подключите путь к видео (можно из VHS Load Video)
3. Вставьте API-ключ WaveSpeed
4. Опишите желаемые звуки и музыку
5. Запустите — результат сохраняется в `ComfyUI/output/`

## Ограничения

- Видео не длиннее 20 секунд
- Промпты до 200 символов каждый
- Стоимость: ~$0.035 за генерацию (WaveSpeed pricing)
